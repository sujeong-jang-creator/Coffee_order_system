우리는 직접 사내카페로 가지 않고도, 전산상으로 커피를 주문할 수 있는 시스템 제작을 의뢰받았습니다.

Mission 02

아래의 조건을 만족시키며 주문을 받아 order_list table을 채우는 make_order 프로시저를 만드세요. * 조건 1. order_id는 sequence를 정의하여 주문 순서대로 값을 채운다. 2. employee_id는 employees 함수의 employee_id와 동일한 형식을 갖는다. 3. date은 시스템의 현재 날짜를 갖는다. 4. 프로시저의 매개변수는 직원 아이디(employee_id), 메뉴 아이디(menu_id), 수량(amount)이다.
답안 작성
모범 답안 확인하기

-- begin절에 잇는 seq_order_no1.nextval이 실행되려면 sequence를 먼저 실행시켜야 합니다. DROP sequence seq_order_no1; CREATE sequence seq_order_no1;

CREATE OR REPLACE procedure make_order( v_emp_id order_list.employee_id%type, v_menu_id order_list.menu_id%type, v_amount order_list.amount%type) IS v_order_id order_list.order_id%type; v_date order_list.order_date%type; v_menu menu.menu_name%type; BEGIN SELECT seq_order_no1.nextval, sysdate INTO v_order_id, v_date FROM dual; INSERT INTO order_list values(v_order_id, v_emp_id, v_menu_id, v_amount, v_date); END; / EXEC make_order(100, 1, 5);

Mission 03

Mission 2에서 주문한 내역에서 메뉴명(menu_id)과 수량(amount)을 수정하는 작업을 수행하는 update_order 프로시저를 만드세요.
답안 작성
모범 답안 확인하기 DROP procedure update_order; CREATE OR REPLACE procedure update_order ( v_order_id order_list.order_id%type, v_menu_id order_list.menu_id%type, v_amount order_list.amount%type ) IS BEGIN UPDATE order_list SET menu_id = v_menu_id, amount = v_amount WHERE order_id = v_order_id END; / EXEC update_order('order_id', 'modified_menu', 'modified_amount');

Mission 04

일일 주문량과 매출을 부서별(department_name)로 정리하여 일일보고서 daily_stats view를 작성하세요. * 조건 1. 일일 주문량은 sum()을 활용하세요. 2. 매출은 sum()과 price * amount를 통해 출력하세요.
답안 작성
모범 답안 확인하기 DROP TABLE tmp; DROP TABLE mmp; CREATE TABLE tmp AS SELECT d.department_name, e.department_id, o.amount, o.menu_id, o.order_date FROM employees e, order_list o, departments d WHERE e.employee_id = o.employee_id AND d.department_id = e.department_id; CREATE TABLE mmp AS SELECT tmp.menu_id, tmp.amount, tmp.department_id, menu.price FROM menu, tmp WHERE menu.menu_id = tmp.menu_id; DROP view daily_stats; CREATE view daily_stats AS SELECT tmp.order_date, tmp.department_name, sum(tmp.amount) AS total_amount, sum(mmp.price * tmp.amount) AS total_price FROM mmp, tmp WHERE mmp.department_id IN (SELECT mmp.department_id FROM mmp) GROUP BY tmp.order_date, tmp.department_name ORDER BY tmp.order_date DESC; SELECT * FROM daily_stats;
